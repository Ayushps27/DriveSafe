# DriveSafe
DriveSafe app
This is an android app based on android N which prevents the user to use mobile phones while driving.
-Uses gps speed to block the mobile display while the device is moving.
-A special parental mode to prevent the user from terminating the app while in use.

****Just an initial prototype****

Additional features that can be added further:
  •A functionality to add live location sharing can be added for enhanced safety of the user.
  •Instead of locking the whole phone, only certain apps can be blocked while the device is in motion.
  •A functionality to read out loud notifications can be added to further increase the usability of the app.
  •Google assistant can be integrated with the app to perform simple tasks such as sending a text or to make a call on speaker while the user is driving. 
  •Navigation through google maps can also be integrated with the app.
